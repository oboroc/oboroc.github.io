RC - text file translation utility

This program was written in Borland Pascal in 1995. At this time I was obsessed by idea to write a MSX emulator. This utility was used to translate tonns of technical documentation from Yamaha MSX-2 russian codetable (custom variation of KOI8-R with double character encoded pseudographic symbols). After a while it showed itself useful also for text translation from early (Florin Ciuperca) to mature (TISH SRL) Romanian DOS code page.

I even launched a very simple MSX-1 game Alibaba on a work-in-progress version of this emu. But after a while I've found on Internet CJB-MSX1 v0.99 emulator, which was much better when what I had, so I dropped MSX emulation project.

You may download RC version 1.3 here.
