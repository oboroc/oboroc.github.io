HYCON - hypergraph conversion program

HYCON is a program to create and manipulate with hypergraph images. This program was written in May, 1997, as a practical realization of an algorithm I developed for hypergraph drawing. A lot of information about is can be found in information file. I was a last year student in the State University of Moldova, and this algorithm and HYCON program formed my diploma work.

Program is written in C++ for DOS using DJGPP 2.01 (gcc 2.7.2.1) development suite. It uses an excelent Allegro 2.10 programming library for graphics, GUI and file i/o. It also uses very nice and easy to use id software installer (the one which was used for DOOM and DOOM2 installation). All graphics used by this program was develped with Deluxe Paint IIe.

HYCON version 1.1 installation is available for download here.

This installarion includes binaries for DOS/DPMI, soruce code, hypergraph examples and diploma work in MS Word 95 document format.
