   HYC0N is a prorgam for playing with hypergraph images. It was developed
   as a part of hypergraph management package, developed by students of the
   State University of Moldova under supervision by associate professor
   Vitalii Voloshin (vol@cinf.usm.md).

   This program can:

        (->     read UDS files (see UDS specs in help)

        (->     construct hypergraph image using linear, B-spline or
                Cat-Mull Rom splines

        (->     perform some relocation operations: place vertices
                on circle, zoom image in and out, fit it to screen

        (->     Save back modified hypergraph structure to UDS file

        (->     Export hypergraph image in PiCTeX (*.PIC) file format,
                usable from TeX compilation systems (emTeX, PCTeX).
                You can also post-edit generated PiCTeX files with
                TeXCAD program from emTeX distribution

        (->     Export hypergraph image in PCX format, usable from a
                wide set of programs (Microsoft Word and Excel etc.)

        (->     Export hypergraph image in TGA format, usable from
                some programs (3D Studio, POVRAY, CompuSHOW etc.)
