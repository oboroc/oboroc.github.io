National symbol support for Adobe PhotoShop 5.5

During my work on Company's web site I meet a serious problem. I wasn't able to create russian, romanian and turkish text in my favorite graphics design software: Adobe PhotoShop 5.5 for Windows. I asked people from FidoNet how they are dealing with this problem and I received two answers:

Eugene Belyaev (2:469/83.210) wrote:

There are several methods to enable cyrillic symbols (sometimes they are working only if combined):

  • Run regedit.exe, go to "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Nls\CodePage" and replace 1252 with "C_1251.NLS" for Windows NT and with "cp_1251.nls" for Windows 9X. Close regedit and reboot the system.
  • Erase file ADMUI3.fon (or ADMUI2.fon) in PhotoShop\Required folder.
  • Erase file AdobeUI.fon in Photoshop folder.

Mike Peleah (2:469/133.33) wrote:

To enable romanian characters please add following section to your win.ini file:

[fontsubstituties]
Tahoma Rom, 0 = Tahoma, 204
Tahoma Rom, 204 = Tahoma, 204
