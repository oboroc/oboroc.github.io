Cyrillics for Slackware Linux console

Credits for the following information goes to the strongest moldovan Linux hacker - Bobb. Thank you, Bobb!

So, you are Slackware Linux user and you wish to enable cyrillics in OS. You will have to modify several files:

In ~/.inputrc
-------------------------------
set meta-flag on
set convert-meta off
set output-meta on
-------------------------------

In the initial local init script:
-------------------------------
/usr/bin/setfont /usr/lib/kbd/consolefonts/Cyr_a8x16
/usr/bin/loadkeys /usr/lib/kbd/keytables/ru1.map
/usr/bin/mapscrn /usr/lib/kbd/consoletrans/koi2alt
-------------------------------
 
In ~/.profile
-------------------------------
echo -ne "\033(K"
-------------------------------
 
In ~/.pinerc
-------------------------------
...
character-set=koi-8r
...
-------------------------------
 
In ~/.joerc : move to the left line with "-asis"
